Connect to GitHub

To publish your changes to the live shop, you need a GitHub Personal Access Token. It's stored only in this browser.

  1. Open github.com/settings/personal-access-tokens/new in a new tab.
  2. Token name: anything (e.g. Musical Magic admin). Expiration: up to a year.
  3. Repository access: Only select repositories → choose richmondbot2000-prog/TShirts.
  4. Repository permissions → Contents: Read and write.
  5. Click Generate token. Copy the token and paste below.
Stored in this browser only. Use "Forget token" to remove it.
